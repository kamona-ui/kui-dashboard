##### To get started:

```bash
# 1. Clone the repository:
git clone https://github.com/kamona-ui/kui-dashboard.git

cd kui-dashboard

# 2. Install the dependencies:
yarn install

# 3. Start the development server:
yarn dev:vue
```

##### Build

```bash
yarn build:vue

# Preview
yarn preview:vue
```
